Music Generation

Create original songs from text prompts. Describe the genre, mood, and instrumentation — the AI handles composition and production.

Custom Lyrics & Style

Full control over lyrics, title, and music style. Write your own lyrics or let AI generate them. Fine-tune genre, BPM, and vocal characteristics.

Voice Personas

Save and reuse vocal styles across songs. Create consistent voices for your entire album with custom persona presets.

Cover & Remix

Transform existing songs into new styles. Turn pop into jazz, rock into acoustic, or any genre cross-over with AI-powered covers and mashups.

Extract vocals, drums, bass, and other instruments from any audio. Isolate individual tracks for remixing and production.

Quick Start

Up and Running in 3 Steps

After installing, follow these steps to connect and start generating music.

Terminal dopehouse-openmic 
# Step 1: Get your API token
# Visit https://platform.acedata.cloud and sign up for an API key

# Step 2: Launch the server
dopehouse-openmic --transport http --port 8000

# Step 3: Connect from your MCP client
# In Claude, Cursor, or VS Code, add an MCP server:
URL: http://localhost:8000/mcp
Authorization: Bearer <your-api-token>
